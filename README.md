<p>隐形球 Invisiballs</p>
<p>Invisiballs是一款可供2到4名玩家参加的本地多人竞技游戏。</p>
<p>在一个自上而下的共享屏幕上玩，每个人都在迷宫中控制一个看不见的球，并且必须捕获一个金色的神器才能赢得比赛。</p>
<p>Invisiballs的设计完全围绕着极简主义世界中的一种新型游戏设置，它简单，具有深刻的学习曲线。</p>
<p>Invisiballs是为喜欢和朋友一起玩竞技游戏的玩家设计的，既紧张又精准。</p>
